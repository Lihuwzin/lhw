README - Guia Completo Hospedagem Minecraft (Pterodactyl)

Como usar (offline / local):
1) Extraia o .zip em uma pasta (ex: C:\ptero-guia)
2) Dê duplo clique em "index.html" (abre direto no navegador)

Observações importantes:
- Funciona totalmente offline (sem dependências externas).
- Os PDFs ficam em: assets/pdfs/
- A fonte Inter (variable) fica em: fonts/
- Em alguns navegadores, SVG externo via <use> pode ser bloqueado ao abrir via file://.
  O site já tem fallback automático para ícones via <img>.

Atalhos:
- Use a busca para pular entre seções
- Clique em "Voltar ao Topo" no canto direito
- Use "Copiar" nos blocos de comando (inclui fallback offline)
